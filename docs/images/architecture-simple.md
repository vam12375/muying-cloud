                      📋 技术栈版本信息
                    ===================
Spring Boot: 3.5.5          MySQL: 8.0.33           JWT: 0.12.6

                      🔄 数据流向说明
                    ===================
1️⃣  用户请求 → API网关 → 路由到具体微服务
2️⃣  微服务之间通过OpenFeign进行RPC调用  
3️⃣  所有服务注册到Nacos，通过Nacos进行服务发现
4️⃣  Sentinel提供限流、熔断、降级保护
5️⃣  Zipkin收集分布式链路追踪数据
6️⃣  核心业务数据存储在MySQL，热点数据缓存到Redis
7️⃣  商品搜索数据存储在Elasticsearch中
8️⃣  支付服务对接支付宝、微信等第三方支付平台
9️⃣  物流服务对接顺丰、圆通、中通等第三方物流API
